Maryland Vaccine Finder App

This web-based application is an exercise in pulling JSON and GeoJSON data via API's from the Maryland State Coronavirus data set. Mainly case and vaccination data were used and called on.

The main goal of this project was to put users in touch with ALL resources within the state of Maryland where they may schedule vaccinations. The application also provided real-time data tracking for vaccinations given and current cases.

The application used HTML5, CSS, Javascript, JQuery, and JSON. An API was configured and used to pull all Json and GeoJSON data from the state data set. Json data was parsed, and results displayed.

Morning Rituals Soap

Morning Rituals Soap is a vendor-facing site, with the purpose of introducing them to handmade soap, which could feature their own coffee/tea products as an ingredient.

This site is based on Wordpress CMS, with some custom CSS for styling. The site is hosted on donbarto.com, with wordpress administration.

Virtually Ready

Virtually Ready was a professional blog I started at the begining of the 2020 COVID-19 global pandemic. It was created as an engagement tool, and proof of concept for a blog targeting professionals forced to work and perform in virtual environments.

This was not my first Wordpress site, but this was the most developed, in terms of content management and heavy reliance on themes.

This project used Wordpress CMS for build, as well as for subsequent blog posts via Wordpress mobile app and site admin. Technologies used were HTML5, CSS3, JavaScript, Wordpress themes and CMS.
